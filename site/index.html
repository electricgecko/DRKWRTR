Statement of purpose.

DRKWRTR invokes an austere, industrial space to write in. It adapts to many dimensions and resolutions. DRKWRTR recognizes Markdown and exports valid HTML.

DRKWRTR saves content locally and automatically. One document per device, per design. If requested, DRKWRTR renders its content to a static website, acessible via permalink. The DRKWRTR functionality is entirely controlled by keyboard shortcuts.
